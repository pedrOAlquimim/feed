# feed

## English

### What is the project?

1º Project of Bootcamp Ignite @Rocketseat. It's the client side of a social media feed, with posts, comments, and likes. To learn the main concepts of React and TypeScript.

### Main technologies

- React
- TypeScript
- ViteJs
